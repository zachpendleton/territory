# Territory

A game server. Fun!

## API

### GET /api/games/{id}

Fetch the status of a previously created game.

### POST /api/games

Create a new Game object.

### POST /api/games/{id}/players

Create a new player inside a game.

### POST /api/games/{id}/moves

Play a tile in an in-progress game.
